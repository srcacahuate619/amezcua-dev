# Amezcua.dev — Soporte Técnico & Desarrollo de Software · Linares, N.L.

Landing page oficial y portafolio de servicios. Este proyecto centraliza mi oferta profesional, abarcando desde el soporte técnico de hardware a nivel local en Linares, N.L., hasta el desarrollo de software B2B y arquitecturas de alto rendimiento.

## 🚀 Servicios Ofrecidos

### 1. Ingeniería de Software y Consultoría B2B
- **Sistemas a la medida:** Creación de aplicaciones escalables.
- **Optimización Logística:** Algoritmos para reducción de costos operativos y rutas.
- **Web3 & HealthTech:** Desarrollo de arquitecturas seguras (Creador de BlackBox Medical, proyecto incubado por Solana Foundation).

### 2. Soporte Técnico Especializado (Clínica de PCs)
- Mantenimiento preventivo y correctivo de equipos de cómputo.
- Ensamble de PCs (Gamer y Workstations).
- Optimización de rendimiento (Migración a SSD, aumento de RAM).

## 🧠 Mi Diferenciador: Rigor Clínico Aplicado al Software
Tras 3 años de formación en medicina clínica, aplico el mismo rigor analítico, precisión y enfoque de "cero fallos" al código y al hardware. Las soluciones no se entregan a medias; se diagnostican y se resuelven de raíz.

## 💻 Stack Tecnológico del Proyecto
- **Framework:** Next.js 14 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** CSS-in-JS (Variables CSS nativas para máxima velocidad y cero dependencias)
- **Despliegue:** Vercel

## ⚙️ Desarrollo Local

Para correr este proyecto en tu máquina:

```bash
npm install
npm run dev
```
